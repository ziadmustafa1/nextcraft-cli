# Contributing to NextCraft

Thanks for helping.

How to start
1. Fork the repo
2. Clone: git clone https://github.com/ziadmustafa1/NextCraft.git
3. Create branch: git checkout -b feat/short-description
4. Install: pnpm install
5. Run example: pnpm --filter nextcraft-example-basic dev
6. Run tests: pnpm test
7. Open PR with clear description and testing steps

Communication
- Open an Issue to discuss big changes
- Use GitHub Discussions for roadmap ideas

Code rules
- Follow existing style and lint rules
- Add tests for new features
- Use conventional commits like feat:, fix:, docs:

We respond within 72 hours.
